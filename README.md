# Py3GPSLogger
Ublox USB GPS Data Logger for Use with Raspberry Pi 4 Running Python 3 Environment within Raspberry Pi OS x64

Materials Required:
1. Raspberry Pi 4 with Raspberry Pi OS x64 Installed
2. Ublox USB GPS
3. 2A Car Supply (USB Type C)
